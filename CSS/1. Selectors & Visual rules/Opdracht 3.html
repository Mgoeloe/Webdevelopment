Opdracht 3

Type selector:
a {color: red;}
Selecteerd het gelijknamige element.

Descendant selector:
ul li {color:red}
Maakt het mogelijk om nested elementen te selecteren.

Class selector:
.class-naam {color:red}
Selecteerd een gelijknamige class, naam wordt zelf verzonnen en meerdere elementen kunnenn worden geselecteerd.
