Cascade:

Zijn de voorrangsregels voor CSS commando's. Als er twee gelijkwaardige commando's zijn, wordt de laatstgebruikte commando gebruikt.

Voorbeeld: In CSS wordt deze tekst rood en blauw gekleurd. Blauw wordt toegepast, omdat dit het laatste commando is.


Inheritance:

Sommige CSS commando's die op een element zijn toegepast hebben ook invloed op de elementen die daarin genest zijn (de child elements) als deze geen specifieke styling krijgen toegepast.
