Opdracht 16

Voor deze opdracht is het handig om in te lezen over methods van String. Aan de linkerkant staat een menubalk. Onder de categorie methods staan een lijst met functies die je met een String kan aanroepen. Maak een eigen voorbeeld van de volgende functies en licht toe wat de functie doet in het blok:

  1. charAt() geeft de letter terug wat met het aangegeven nummer wordt geselecteerd, begint vanaf 0 met tellen.

    Voorbeeld:
    let test = "qwertyuiop"
    let result = test.charAt(5)
    console.log(result);

  2. concat() kan twee strings samenvoegen.

    Voorbeeld:
    let string1 = "goede";
    let string2 = "morgen";
    let result = string1.concat(string2);
    console.log(result);

  3. slice() kan een deel van een string extraheren.

    Voorbeeld:
    let string1 = "goedemorgen";
    let result = string1.slice(5);
    console.log(result);

  4. split() wordt gebruikt om een string te splitsen.

    Voorbeeld:
    let string1 = "goedemorgen";
    let result = string1.split("");
    console.log(result);

  5. substr() extraheert een deel van een string. De values geven het begin en einde aan.

    Voorbeeld:
    let string1 = "goedemorgen";
    let result = string1.substr(0,5);
    console.log(result);
