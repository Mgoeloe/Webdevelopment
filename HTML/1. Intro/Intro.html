Vraag 1

Internet is het fysieke netwerk wat miljoenen computers met elkaar verbind.

World wide web is een toepassing op het internet die webpagina's aan elkaar verbind via links.

Vraag 2

HTML is om de structuur en inhoud van een webpagina te bepalen, CSS is voor de opmaak en Javascript is om interactie toe te passen.

Vraag 3

Een tag is een opdracht die door een browser kan worden geïmplementeerd en een element is de tag + inhoud.
Je opent een tag door deze in te sluiten in kleiner-dan/groter-dan tekens en sluit deze door deze handeling te herhalen met een slash achter het kleiner-dan teken.
Voorbeeld: <p> blablabla </p>

Vraag 4

<html> geeft het protocol aan.
<head> geeft de metadata aan.
<body> is de (zichtbare) inhoud van de pagina.

Vraag 6

Structuur is belangrijk, zodat pagina's correct en overzichtelijk worden weergegeven.
De weergegeven code heeft geen goede structuur, omdat de tekst als één lange zin weergegeven zal worden.

Vraag 7

Een block element begint altijd op een nieuwe regel en gebruikt de volledige breedte van een venster.
Een inline element start niet op een nieuwe regel en gebruikt alleen de ruimte die nodig is om de inhoud weer te geven.

Vraag 8

Nesting is het gebruiken van een element binnen een ander element.
Deze tekst is dikgedrukt en deze tekst is dikgedrukt en schuin.
