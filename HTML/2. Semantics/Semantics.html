Vraag 9

Bij semantische elementen is de naam van een element relevant aan de inhoud.
Het is belanrijk, omdat je code overzichtelijker wordt en efficiënter kan worden gelezen. Een voordeel is ook dat de browser je code beter kan interpreteren.

Vraag 10

<header> Titel en logo van de pagina
<footer> Contactinformatie
<main> De content van de pagina
<nav> Een navigatiebar of drop-down menu om efficiënt door de website te kunnen navigeren
<section> De pagina in verschillende secties verdelen
<aside> Een sidebar maken naast de hoofdcontent
<article> Losse artikels toevoegen aan een pagina
